# ModedLitterRobotWasteIndicator

These files should be dropped directly into an Adafruit Feather STM32F405 Express with a connected Adafruit VL53L0X Time of Flight Distance Sensor
This is for a modified Litter Robot 3, that has the bottom cut out so the waste drops into a 5 gallon bucket.
The sensor determines the distance from the bottom of the robot to the bottom of the 5 gallon bucket to determine how full of waste it is.
